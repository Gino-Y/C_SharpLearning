# 一维数组

> 对应 `Class01/Lesson15.cs`

```csharp
int[] intArray = new int[3];   // 长度 3，默认值全是 0
intArray[2] = 10;              // 下标从 0 开始

for (int i = 0; i < intArray.Length; i++)   // .Length 是元素个数
{
    Console.WriteLine(intArray[i]);
}

foreach (var item in intArray)   // 迭代遍历
{
    Console.WriteLine(item);
}
```

三种初始化写法（等价）：

```csharp
int[] a = new int[] { 1, 2, 3, 4, 5 };
int[] b = new int[5] { 1, 2, 3, 4, 5 };
int[] c = { 1, 2, 3, 4, 5 };
```

## 要点

- 下标从 **0** 开始；`数组.Length` 得到元素个数。
- 数值类型数组默认值是 `0`，引用类型是 `null`。
- 数组**长度固定**，创建后不能增删元素（需要变长用 `List<T>`）。
