# 程序结构与输出
[下一篇：变量与条件分支（if）](./变量与条件分支.md)

> [!NOTE]
> 对应 `Class01/Lesson01.cs`、`Lesson02.cs`。本篇是全部笔记的起点。

```csharp
namespace CSharpLearning.Class01;   // 命名空间（文件范围写法，末尾带分号）

public class Lesson02
{
    public static void Run()
    {
        Console.WriteLine("Hello, World!");
    }
}
```

## 要点

- **命名空间 `namespace`**：一组类的容器。两种写法**不能混用**：
  - 文件范围（推荐）：`namespace X;`（分号结尾，**不加** `{}`）
  - 块范围：`namespace X { ... }`（**不加**分号）
- **类 `class`**：代码的基本组织单位。
- **`Console.WriteLine()`**：输出一行并换行；`Console.Write()` 不换行。
- 项目开启了 `ImplicitUsings`，用 `Console` 不需要手写 `using System;`。
