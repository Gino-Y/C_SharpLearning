# 二维数组与交错数组

> 对应 `Class01/Lesson16.cs`

## 二维数组 `int[,]`（方阵，每行列数相同）

```csharp
int[,] arrays2 = { {1, 2}, {3, 4}, {5, 6} };   // 3 行 2 列

for (int i = 0; i < arrays2.GetLength(0); i++)   // 第0维 = 行数
{
    for (int j = 0; j < arrays2.GetLength(1); j++)   // 第1维 = 列数
    {
        Console.Write(arrays2[i, j] + " ");    // 一个方括号，逗号分隔
    }
    Console.WriteLine();
}
```

## 交错数组 `int[][]`（数组的数组，每行列数可不同）

```csharp
int[][] arrays3 = new int[3][];
arrays3[0] = new int[] { 1, 2 };
arrays3[1] = new int[] { 3, 4, 5 };   // 这一行 3 列
arrays3[2] = new int[] { 6 };

for (int i = 0; i < arrays3.Length; i++)
{
    for (int j = 0; j < arrays3[i].Length; j++)   // 每行长度可能不同
    {
        Console.Write(arrays3[i][j] + " ");    // 两个方括号
    }
    Console.WriteLine();
}
```

## 关键区别（易错点）

| | 二维数组 `int[,]` | 交错数组 `int[][]` |
|---|---|---|
| 形状 | 方阵，每行等长 | 每行可不等长 |
| 取元素 | `a[i, j]`（逗号） | `a[i][j]`（两个方括号） |
| 各维长度 | `GetLength(0)` / `GetLength(1)` | `a.Length` / `a[i].Length` |

> 陷阱：对 `int[,]` 用 `a[i]`（单下标）或 `a.Length`（当行数）都是错的——`.Length` 返回**元素总数**，行数要用 `GetLength(0)`。

## 维度概念

- 声明 `int[3, 2]`：逗号左边是**第 0 维**（行=3），右边是**第 1 维**（列=2）。
- 编号从 0 开始，`GetLength(0)` 拿第 0 维长度。
- 「先行后列」的习惯与 Python/NumPy 一致；区别是取元素 C# 二维用 `[i, j]`，交错/嵌套列表用 `[i][j]`。
