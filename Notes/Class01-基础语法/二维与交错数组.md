# 二维数组与交错数组

[上一篇：一维数组](./一维数组.md) · [下一篇：方法与方法重载](./方法与方法重载.md)

> [!NOTE]

## 二维 `int[,]` vs 交错 `int[][]`

```csharp
int[,] a = { {1, 2}, {3, 4} };           // 方阵；取 a[i, j]；维长 GetLength(0/1)
int[][] b = new int[2][];                 // 锯齿；取 b[i][j]；行长 b[i].Length
b[0] = new[] { 1, 2 }; b[1] = new[] { 3, 4, 5 };
```

| | `int[,]` | `int[][]` |
|---|---|---|
| 形状 | 每行等长 | 每行可不等长 |
| 取元素 | `a[i, j]` | `a[i][j]` |
| 长度 | `GetLength(维)` | `a.Length` / `a[i].Length` |

> [!WARNING]
> 对 `int[,]` 用 `a[i]` 或把 `.Length` 当行数 → 错（`.Length`=元素总数，行数用 `GetLength(0)`）。

> [!TIP]
> 先行后列与 Python 一致；二维用 `[i,j]`，交错/嵌套用 `[i][j]`。
