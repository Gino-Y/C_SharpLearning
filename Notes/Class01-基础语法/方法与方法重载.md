# 方法与方法重载
[上一篇：二维与交错数组](./二维与交错数组.md)

> [!NOTE]
> 对应 `Class01/Lesson17.cs`
> 前置：[程序结构与输出](./程序结构与输出.md) · 相关：[运算符重载](../Class02-面向对象/运算符重载.md)、[拓展方法](../Class02-面向对象/拓展方法.md)

```csharp
static void Speak()          // 无参、无返回值
{
    Console.WriteLine("Hello World!");
}

static int Add(int a, int b)         // 返回 int
{
    return a + b;
}

static float Add(float a, float b = 2)   // 方法重载 + 默认参数
{
    return a + b;
}
```

## 要点

- 方法签名：`修饰符 返回类型 方法名(参数列表)`；无返回值用 `void`。
- **形参**（定义时的参数）可以有**默认值**（`b = 2`），调用时可省略。
- **方法重载 (overload)**：同名方法，只要满足以下之一即可：
  1. 参数**个数**不同
  2. 参数**类型**不同
  （注意：仅返回类型不同**不算**重载。）
- 编译器根据传入的实参自动选择匹配的重载版本。
