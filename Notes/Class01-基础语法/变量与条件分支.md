# 变量与条件分支（if）

> 对应 `Class01/Lesson11.cs`

```csharp
int a = 6;
if (a < 10 && a > 5)
{
    Console.WriteLine("a is between 5 and 10");
}
else
{
    Console.WriteLine("a is not between 5 and 10");
}
Console.WriteLine($"a = {a}");   // $"" 是字符串插值
```

## 要点

- 变量要**先声明类型**：`int a = 6;`。
- 逻辑运算符：`&&`（且）、`||`（或）、`!`（非）。
- **字符串插值** `$"...{变量}..."`：把变量嵌进字符串，类似 Python 的 f-string。
- `if / else if / else` 结构，条件用 `()` 包起来，body 用 `{}`。
