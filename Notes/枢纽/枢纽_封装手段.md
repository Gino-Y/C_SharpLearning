# 枢纽 · 封装手段

> [!NOTE]
> 把「怎么把数据与用法藏进类型边界」串起来：属性管字段、索引器管容器访问、内部类表亲密结构、拓展/运算符扩展用法。

## 定义链

字段私有 → **属性**控制读写 → **索引器**用 `[]` 访问内部集合 → **内部类**表达强从属 → **拓展方法**不改源码挂方法 → **运算符重载**让对象能算。

## 核心篇目

- [**成员属性**](../Class02-面向对象/成员属性.md)：get/set、value、private set
- [**索引器**](../Class02-面向对象/索引器.md)：`this[...]`，规则自己定
- [**内部类与分部类**](../Class02-面向对象/内部类与分部类.md)：包裹者点出自己；partial 拆申明
- [**拓展方法**](../Class02-面向对象/拓展方法.md)：静态类 + `this` 首参
- [**运算符重载**](../Class02-面向对象/运算符重载.md)：`public static operator`

## 口诀

> [!TIP]
> **藏数据用属性；扮数组用索引器；亲儿子用内部类；外挂用拓展；能算靠 operator。**
