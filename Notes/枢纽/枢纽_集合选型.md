# 枢纽 · 集合选型

> [!NOTE]
> 按访问模式选容器：按下标、栈、队列、中间插删、按键查找；细节见各篇，对比见总览。

## 定义链

需要可变长序列 → **List** → 只要 LIFO → **Stack** → 只要 FIFO → **Queue** → 频繁中间插删 → **LinkedList** → 按键 O(1) → **Dictionary** → 拿不准先看 **集合总览**。
