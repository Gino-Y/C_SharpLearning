## Lesson01 / 02：程序结构与输出

```csharp
namespace CSharpLearning.Class01;   // 命名空间（文件范围写法，末尾带分号）

public class Lesson02
{
    public static void Run()
    {
        Console.WriteLine("Hello, World!");
    }
}
```

要点：

- **命名空间 `namespace`**：一组类的容器。两种写法**不能混用**：
  - 文件范围（推荐）：`namespace X;`（分号结尾，**不加** `{}`）
  - 块范围：`namespace X { ... }`（**不加**分号）
- **类 `class`**：代码的基本组织单位。
- **`Console.WriteLine()`**：输出一行并换行；`Console.Write()` 不换行。
- 因为项目开启了 `ImplicitUsings`，用 `Console` 不需要手写 `using System;`。

## Lesson11：变量与 if 分支

```csharp
int a = 6;
if (a < 10 && a > 5)
{
    Console.WriteLine("a is between 5 and 10");
}
else
{
    Console.WriteLine("a is not between 5 and 10");
}
Console.WriteLine($"a = {a}");   // $"" 是字符串插值
```

要点：

- 变量要**先声明类型**：`int a = 6;`。
- 逻辑运算符：`&&`（且）、`||`（或）、`!`（非）。
- **字符串插值** `$"...{变量}..."`：把变量嵌进字符串，类似 Python 的 f-string。
- `if / else if / else` 结构，条件用 `()` 包起来，body 用 `{}`。

## Lesson12：switch 分支

```csharp
int a = 9;
switch (a)
{
    case 0:
        Console.WriteLine("a == 0");
        break;
    case 1:
        Console.WriteLine("a == 1");
        break;
    default:
        Console.WriteLine("a == ?");
        break;
}
```

要点：

- 每个 `case` 后**必须有 `break;`**（否则编译错误，C# 不允许自动“贯穿”）。
- `default` 处理所有未匹配的情况。
- 多个固定取值的判断，用 `switch` 比一长串 `if-else` 清晰。

## Lesson13：while / do-while 循环

```csharp
int num = 6;
do
{
    Console.WriteLine($"num = {num}");
    num++;
} while (num < 5);        // 注意结尾有分号
```

要点：

- `while (条件) { }`：**先判断**再执行，条件不成立一次都不跑。
- `do { } while (条件);`：**先执行一次**再判断，所以即使 `num=6` 不满足 `<5`，也会打印一次。
- `do-while` 结尾的分号 `;` 不能忘。

## Lesson14：for / foreach 循环

```csharp
// for：适合按下标循环
for (int i = 0; i < 5; i++)
{
    Console.WriteLine($"i = {i}");
}

// foreach：适合遍历集合中每个元素
string[] names = { "Alice", "Bob", "Charlie" };
foreach (string name in names)
{
    Console.WriteLine($"name = {name}");
}
```

要点：

- `for (初始化; 条件; 步进)`：能拿到下标 `i`，适合需要索引的场景。
- `foreach (类型 变量 in 集合)`：直接取每个元素，**拿不到下标**，写法更简洁。
- 只读取元素 → 用 `foreach`；需要下标或修改元素 → 用 `for`。

## Lesson15：一维数组

```csharp
int[] intArray = new int[3];   // 长度 3，默认值全是 0
intArray[2] = 10;              // 下标从 0 开始

for (int i = 0; i < intArray.Length; i++)   // .Length 是元素个数
{
    Console.WriteLine(intArray[i]);
}

foreach (var item in intArray)   // 迭代遍历
{
    Console.WriteLine(item);
}
```

三种初始化写法（等价）：

```csharp
int[] a = new int[] { 1, 2, 3, 4, 5 };
int[] b = new int[5] { 1, 2, 3, 4, 5 };
int[] c = { 1, 2, 3, 4, 5 };
```

要点：

- 下标从 **0** 开始；`数组.Length` 得到元素个数。
- 数值类型数组默认值是 `0`，引用类型是 `null`。

## Lesson16：二维数组与交错数组

### 二维数组 `int[,]`（方阵，每行列数相同）

```csharp
int[,] arrays2 = { {1, 2}, {3, 4}, {5, 6} };   // 3 行 2 列

for (int i = 0; i < arrays2.GetLength(0); i++)   // 第0维 = 行数
{
    for (int j = 0; j < arrays2.GetLength(1); j++)   // 第1维 = 列数
    {
        Console.Write(arrays2[i, j] + " ");    // 一个方括号，逗号分隔
    }
    Console.WriteLine();
}
```

### 交错数组 `int[][]`（数组的数组，每行列数可不同）

```csharp
int[][] arrays3 = new int[3][];
arrays3[0] = new int[] { 1, 2 };
arrays3[1] = new int[] { 3, 4, 5 };   // 这一行 3 列
arrays3[2] = new int[] { 6 };

for (int i = 0; i < arrays3.Length; i++)
{
    for (int j = 0; j < arrays3[i].Length; j++)   // 每行长度可能不同
    {
        Console.Write(arrays3[i][j] + " ");    // 两个方括号
    }
    Console.WriteLine();
}
```

### 关键区别（易错点）

| | 二维数组 `int[,]` | 交错数组 `int[][]` |
|---|---|---|
| 形状 | 方阵，每行等长 | 每行可不等长 |
| 取元素 | `a[i, j]`（逗号） | `a[i][j]`（两个方括号） |
| 各维长度 | `GetLength(0)` / `GetLength(1)` | `a.Length` / `a[i].Length` |

> 陷阱：对 `int[,]` 用 `a[i]`（单下标）或 `a.Length`（当行数）都是错的——`.Length` 返回**元素总数**，行数要用 `GetLength(0)`。

## Lesson17：方法与方法重载

```csharp
static void Speak()          // 无参、无返回值
{
    Console.WriteLine("Hello World!");
}

static int Add(int a, int b)         // 返回 int
{
    return a + b;
}

static float Add(float a, float b = 2)   // 方法重载 + 默认参数
{
    return a + b;
}
```

要点：

- 方法签名：`修饰符 返回类型 方法名(参数列表)`；无返回值用 `void`。
- **形参**（定义时的参数）可以有**默认值**（`b = 2`），调用时可省略。
- **方法重载 (overload)**：同名方法，只要满足以下之一即可：
  1. 参数**个数**不同
  2. 参数**类型**不同
  （注意：仅返回类型不同**不算**重载。）
- 编译器根据传入的实参自动选择匹配的重载版本。

## 与 Python 快速对照

| 概念 | Python | C# |
|------|--------|-----|
| 输出 | `print()` | `Console.WriteLine()` |
| 变量 | `a = 6`（动态类型） | `int a = 6;`（静态类型） |
| 字符串格式化 | f-string `f"{a}"` | 插值 `$"{a}"` |
| 分支 | `if/elif/else` | `if/else if/else`、`switch` |
| 遍历 | `for x in list` | `foreach (var x in list)` |
| 数组 | `list`（可变长） | `int[]`（定长）、`List<T>`（变长） |
| 函数 | `def f():` | `static 返回类型 F() {}` |
| 重载 | 不支持（靠默认参数） | 支持方法重载 |
