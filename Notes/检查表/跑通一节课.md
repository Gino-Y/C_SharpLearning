# 检查表 · 跑通一节课

> [!NOTE]
> 行动清单，不是知识点推导。规则见 [AGENTS.md](../../AGENTS.md)。

## 步骤

1. 打开课例 `.cs`，确认入口是 `public static void Run()`（不要另起 `Main`）。
2. 在 `Program.cs` 的 `Main` 里调用对应 `Xxx.Run()`（命名空间对得上）。
3. 终端执行：`dotnet run`
4. 对照笔记预期输出；若异常，先看笔记 WARNING 踩坑再查源码。
5. 跑完后把 `Program.cs` 调回你当前在学的那一节，避免误跑旧课。
