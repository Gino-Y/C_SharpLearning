> [!IMPORTANT]
> 笔记规则**唯一真源**：[AGENTS.md](../AGENTS.md)。本页保留**本库标准要求一览**，便于与其它笔记库（如 TLS_UGUI）对照差异；细则仍以 AGENTS 为准。

## 标准要求一览（本库）

| 维度 | 本库要求 |
|------|----------|
| 规则真源 | 仓库根 `AGENTS.md` |
| 层级 | L1 README/枢纽 → L2 速查卡 → L3 篇目；**禁 L4** |
| 导航原则 | 本级详述 + 子级简报 |
| 单篇预算 | L3 ≤ **60 行**（上限不是目标；图/音频行可不计）；L1/L2 索引豁免 |
| 代码片段 | 核心 **10–15 行**，注明「完整代码见 `XXX.cs`」 |
| 密度 | 宁表勿串；踩坑「症状→原因→修法」；Python 对照 ≤3 行 |
| 链接主链 | **markdown 相对路径**（保 GitHub）；不强制 wikilink |
| 溯源 | frontmatter `对应课件:` → 课例 `.cs`（无 `Sources/` 层） |
| 序链 | 同模块/同子簇上下篇 |
| 跨子域 | [枢纽](./枢纽/00_枢纽索引.md)（封装/类型/集合/委托） |
| 行动层 | [检查表](./检查表/跑通一节课.md) |
| 标签 | 1 类型 +（非索引）1 主题；禁正文 `#标签` |
| 类型词表 | `知识点` · `练习` · `实战` · `索引` |
| 主题词表 | 语法基础 / 流程控制 / 数组 / 方法 / 类与对象 / 类型系统 / 继承多态 / 泛型 / 集合 / 委托 / 工具链 |
| 文件夹命名 | 保持 `Class01-基础语法/` 等现名（不强制 `编号_主题/`） |

### 与 TLS_UGUI 笔记标准的主要差异

| 点 | 本库（CSharpLearning） | TLS_UGUI |

## 组织方式（摘要）

- **一个知识点 = 一个 L3 文件**，放在对应模块子文件夹。
- **L2 速查卡**（`01-基础语法.md`、`02-面向对象.md`）：本级脉络 + 各篇一行简报，不堆正文。
- **跨子域复习**走 [枢纽索引](./枢纽/00_枢纽索引.md)；**动手自查**用 [检查表](./检查表/跑通一节课.md)。
- 新建笔记参照 [模板](./模板/知识点笔记.md)。

## 标签词表（受控）

新增标签必须先改本节与 [AGENTS.md](../AGENTS.md) O1。

- 每篇恰好 1 个 `类型/`；非索引篇恰好 1 个 `主题/`；索引不打主题。禁止正文 `#标签`。
- **类型**：`类型/知识点` · `类型/练习` · `类型/实战` · `类型/索引`
- **主题**：`主题/语法基础` · `主题/流程控制` · `主题/数组` · `主题/方法` · `主题/类与对象` · `主题/类型系统` · `主题/继承多态` · `主题/泛型` · `主题/集合` · `主题/委托` · `主题/工具链`
