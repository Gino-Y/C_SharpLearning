# 委托练习：智能家居遥控器

> 对应 `Class02/Lesson19_Practice.cs`。知识点见 [delegate 委托](./委托.md)。
> 要求自己写代码实现，不看 Lesson19b 抄。

## 场景

家里有一个遥控器，按一下「回家模式」，已注册的电器各自执行自己的动作。

## 要求

### 1. 电器类（三个，普通类即可）

- `Light`（灯）：方法 `TurnOn()`，打印 `客厅灯亮了`
- `AirConditioner`（空调）：方法 `StartCool()`，打印 `空调制冷 26 度`
- `Speaker`（音箱）：方法 `PlayMusic()`，打印 `音箱开始放歌`

> 考点一：三个方法**名字各不相同**——但只要签名都是「无参、无返回值」，就都能装进同一个委托。

### 2. 遥控器类 `RemoteControl`

- 声明委托类型 `GoHomeHandler`（无参无返回）
- 一个委托变量 `OnGoHome`（注意可空 `?`）
- 方法 `PressButton()`：打印 `按下回家键 ------`，然后**安全地**调用委托

> 考点二：没人注册时按按钮，程序不能崩（判空调用）。

### 3. `Run()` 方法

1. 创建遥控器和三个电器
2. **先直接按一次按钮**（没人注册，验证不报错）
3. 三个电器的方法都 `+=` 注册
4. 再按一次 → 三个电器都响应
5. `-=` 取消注册空调
6. 再按一次 → 只有灯和音箱响应

> 考点三：`-=` 取消注册。

## 预期输出

```
按下回家键 ------
按下回家键 ------
客厅灯亮了
空调制冷 26 度
音箱开始放歌
按下回家键 ------
客厅灯亮了
音箱开始放歌
```
