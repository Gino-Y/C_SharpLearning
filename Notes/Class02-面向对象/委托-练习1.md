# 委托练习：智能家居遥控器

[上一篇：委托](./委托.md) · [下一篇：委托练习 2](./委托-练习2.md)

> [!NOTE]
> 对应 `Class02/Lesson19_Practice.cs`。自己实现，勿抄 Lesson19b。
> 前置：[委托](./委托.md) · 相关枢纽：[委托体系](../枢纽/枢纽_委托体系.md)

## 要求

1. **电器**：`Light.TurnOn` → `客厅灯亮了`；`AirConditioner.StartCool` → `空调制冷 26 度`；`Speaker.PlayMusic` → `音箱开始放歌`（签名均可无参无返回）。
2. **遥控器** `RemoteControl`：委托 `GoHomeHandler`；变量 `OnGoHome?`；`PressButton()` 打印后安全调用。
3. **Run**：先空按（不崩）→ 三个 `+=` → 再按全响应 → `-=` 空调 → 再按只剩灯+音箱。

## 预期

```
按下回家键 ------
按下回家键 ------

空调制冷 26 度
音箱开始放歌
按下回家键 ------
客厅灯亮了
音箱开始放歌
```
