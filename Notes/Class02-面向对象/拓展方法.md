# 拓展方法（扩展方法）

[上一篇：内部类与分部类](./内部类与分部类.md) · [下一篇：运算符重载](./运算符重载.md)

> [!NOTE]
> 对应 `TLS_HX/Lesson09_封装_拓展方法.cs`。给现有类型外挂方法，不改源码。
> 前置：[static](./static静态成员.md) · [方法与方法重载](../Class01-基础语法/方法与方法重载.md) · 相关枢纽：[封装手段](../枢纽/枢纽_封装手段.md)

知识链：静态类里的静态方法 + 首参 `this` → 调用从 `Tools.X(x)` 变成 `x.X()`。

## 语法

`static 返回值 名(this 拓展类型 参数, …)` — 完整见课例。

```csharp
static class Tools {
    public static void SpeakValue(this int value) => Console.WriteLine(value);
}
10.SpeakValue();           // 首参不传，接收者即 this
test.Func2();              // 与成员同名 → 成员优先
Tools.Func2(test);         // 静态调法仍可调到拓展
```

## 规则与注意

1. 必须在静态类；必须是静态方法；首参是目标且用 `this`。
2. 可有返回值与额外参数。本质语法糖；LINQ 的 `Where`/`Select` 即官方拓展。

> [!TIP]
> Python ≈ monkey patch；C# 是编译期检查的安全外挂。
