# 运算符重载

[上一篇：拓展方法](./拓展方法.md)

> [!NOTE]
> 对应 `TLS_HX/Lesson10_封装_运算符重载.cs`。教会编译器 `p1 + p2` 怎么算。
> 前置：[方法与方法重载](../Class01-基础语法/方法与方法重载.md) · [static](./static静态成员.md) · 相关枢纽：[封装手段](../枢纽/枢纽_封装手段.md)

## 语法

`public static 返回类型 operator 运算符(参数列表)` — 完整见课例。

```csharp
public static Point operator +(Point p1, Point p2) {
    return new Point { x = p1.x + p2.x, y = p1.y + p2.y };
}
public static Point operator +(Point p, int value) { /* ... */ }
public static Point operator +(int value, Point p) => p + value; // 顺序不自动交换
```

## 特点与注意

1. 必须 `public static`；返回值写在 `operator` 前；逻辑自定义。
2. `==`/`!=`、`</>`、`<=/>=` 成对实现；可多签名重载；参数不能用 `ref`/`out`。

> [!WARNING]
> - 漏 `static` → must be static and public。
> - `new Point(a,b)` 无带参构造 → 补构造（自定义后无参也要手写）。
> - 参数/局部都叫 `p` → 结果改名 `result`。

> [!TIP]
> Python：`operator +` ≈ `__add__`；`(int, Point)` ≈ `__radd__`。
