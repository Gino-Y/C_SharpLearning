# 运算符重载

> 对应 `TLS_HX/Lesson10_封装_运算符重载.cs`（唐老师核心课程）。运算符重载 = **让自定义类和结构体对象可以进行运算**——教会编译器 `p1 + p2` 该怎么算。

## 基本语法

使用关键字 `operator`：

```
public static 返回类型 operator 运算符(参数列表)
```

核心片段（完整代码见 `TLS_HX/Lesson10_封装_运算符重载.cs`）：

```csharp
class Point
{
    public int x;
    public int y;

    public static Point operator +(Point p1, Point p2)   // Point + Point
    {
        Point result = new Point();
        result.x = p1.x + p2.x;
        result.y = p1.y + p2.y;
        return result;
    }

    public static Point operator +(Point p, int value)   // Point + int（略）
    { /* ... */ }

    public static Point operator +(int value, Point p)   // int + Point：顺序不自动交换，
    {                                                    // p+10 和 10+p 是两个签名
        return p + value;                                // 复用上面的重载，不重抄逻辑
    }
}

Point p3 = p1 + p2;   // 编译器翻译成 Point.operator+(p1, p2)
```

## 特点

1. **一定是一个公共的静态方法**（`public static` 缺一不可）
2. **返回值写在 `operator` 前**
3. **逻辑处理自定义**

## 注意

1. **条件运算符需要成对实现**：重载 `==` 必须配 `!=`，`<` 配 `>`，`<=` 配 `>=`
2. **一个符号可以多个重载**（参数类型/顺序不同即可）
3. **不能使用 `ref` 和 `out`** 修饰参数

## 踩过的坑（Lesson10 实操）

- 漏写 `static` → `must be declared static and public` → "+"不属于任何对象，是类级静态调用。
- `new Point(a, b)` 报参数不符 → 类里没写带参构造 → 补带参构造，且无参的也要手动补回。
- 参数与局部变量都叫 `p` → `Conflicting variable` → 结果对象改名 `result`。

## Python 对照

`operator +` ≈ `__add__`；C# 的 `(int, Point)` 重载 ≈ Python 的 `__radd__`（右加），思想一致。
