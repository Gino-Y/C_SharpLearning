# struct 结构体

[上一篇：值类型 vs 引用类型](./值类型与引用类型.md) · [下一篇：enum 枚举](./枚举.md)

> [!NOTE]
> 对应 `Class02/Lesson17.cs`。struct = 自定义值类型。
> 前置：[值类型 vs 引用类型](./值类型与引用类型.md) · 相关枢纽：[类型系统](../枢纽/枢纽_类型系统.md)

```csharp
struct Point { public int X, Y; public Point(int x, int y) { X = x; Y = y; } }
Point p2 = p1; p2.X = 100;   // p1.X 不变（复制一份）
```

| | struct | class |
|---|---|---|
| 赋值 | 复制值 | 复制引用 |
| 继承 | 否 | 是 |
| 接口 | 可实现 | 可 |
| null | 否（除非 `T?`） | 可 |
| 适合 | 小而简单的值 | 默认选它 |

- 可实现接口，不能继承类；无 virtual/override 那套。
- struct 是元素模板，List 是容器：`List<Point>`。

> [!TIP]
> Python `namedtuple`/`dataclass` 仍是引用语义，无「赋值复制」。
