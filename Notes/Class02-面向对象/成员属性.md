# 成员属性（get / set）

> 对应 `TLS_HX/Lesson05_成员属性.cs`（唐老师核心课程）。属性 = **给私有字段包一层可控的门**，外部看着像变量，实际走的是 get/set 两个方法。前置知识：[类与对象](./类与对象.md)（那里用的自动属性是本篇的简写形态）。

## 基本概念

1. **用于保护成员变量**（字段设为 `private`，外部只能通过属性进出）
2. **为获取和赋值添加逻辑处理**（校验、加密等）
3. **解决 3P 的局限性**：
   - `public` — 内外访问
   - `private` — 内部访问
   - `protected` — 内部和子类访问
   - 3P 只能"全开或全关"；属性可以做到**只能获取不能修改**，或者**只能修改不能获取**

## 基本语法

```
访问修饰符 属性类型 属性名
{
    get{}
    set{}
}
```

## 案例代码（TLS_HX/Lesson05）

```csharp
class Person
{
    private string name;    // 字段：私有，藏在类内部
    private int money;

    // 属性命名规则：首字母大写（字段 name → 属性 Name）
    public string Name
    {
        get
        {
            return name;              // 获取前可以加逻辑
        }
        set
        {
            if (value.Length > 10)    // value 是关键字：外部赋过来的那个值
            {
                Console.WriteLine("名字太长，请重新输入");
                return;
            }
            name = value;
        }
    }

    // get/set 里做加密解密：外部完全无感知
    public int Money
    {
        get { return money - 5; }     // 解密
        set
        {
            if (value < 0) { Console.WriteLine("钱不能小于0"); return; }
            money = value + 5;        // 加密
        }
    }
}

// 使用：像变量一样读写，实际走的是 get/set
Person person = new Person();
person.Name = "张三";       // 触发 set，"张三" 就是 value
Console.WriteLine(person.Name);   // 触发 get
```

## get 和 set 前可以加访问修饰符

```csharp
public int Age { get; private set; }   // 外部能查不能改
```

三条规则：

1. **默认不加**，使用属性声明时的访问权限
2. **加的访问修饰符要低于属性的访问权限**（属性 public，set 可以 private；反过来不行）
3. **不能让 get 和 set 的访问权限都低于属性的权限**（那属性本身的 public 就没意义了）

注意：`private set` 后，**类外部赋值会直接编译报错**——保护是编译期的，不是运行时报警。

## get 和 set 可以只有一个

```csharp
public bool Sex
{
    get => sex;    // 只读属性；只有一个时没必要在前面加访问修饰符
}
```

- 一般情况只会出现**只有 get 没有 set** 的属性（只读常见，只写罕见）
- `get => sex;` 是表达式简写，等价于 `get { return sex; }`

## 与自动属性的关系

```csharp
public int Age { get; set; }              // 自动属性：编译器暗中生成私有字段+直进直出的get/set
public int Age { get; private set; }      // 外部能查不能改，且无特殊处理 → 直接用自动属性最省事
```

**选择原则**：get/set 里不需要任何逻辑 → 自动属性；需要校验/加密等逻辑 → 本篇的完整写法。

## Python 对照

```python
class Person:
    @property
    def name(self):          # ≈ get
        return self._name

    @name.setter
    def name(self, value):   # ≈ set
        if len(value) > 10: return
        self._name = value
```

思想一致：外部 `person.name = "张三"` 像操作变量，内部走方法。C# 的 `value` 关键字 ≈ Python setter 的 `value` 参数。
