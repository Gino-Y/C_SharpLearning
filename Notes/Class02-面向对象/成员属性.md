# 成员属性（get / set）

[上一篇：static 静态成员](./static静态成员.md) · [下一篇：索引器](./索引器.md)

> [!NOTE]
> 对应 `TLS_HX/Lesson05_成员属性.cs`。属性 = 给私有字段包可控的门。
> 前置：[类与对象](./类与对象.md) · 相关枢纽：[封装手段](../枢纽/枢纽_封装手段.md)

## 概念

1. 保护字段（`private` + 属性进出）
2. get/set 可加逻辑（校验、加密）
3. 突破 3P「全开/全关」：可只读或只写

## 语法

`访问修饰符 类型 名 { get{} set{} }` — 完整见课例。

```csharp
private string name;
public string Name {
    get { return name; }
    set { if (value.Length > 10) return; name = value; }  // value = 外部赋值
}
```

## 访问器与只读

```csharp
public int Age { get; private set; }   // 外能查不能改
```

- 访问器修饰符须**低于**属性权限；不能 get/set **都**低于属性。
- 可只有 get（只读）；`get => sex;` 是简写。
- `{ get; set; }` 自动属性 = 无逻辑时用；要校验用完整写法。

> [!WARNING]
> `private set` 后类外赋值 → CS0272（编译期保护）。

> [!TIP]
> Python：`@property` / `@name.setter`；`value` ≈ setter 参数。
