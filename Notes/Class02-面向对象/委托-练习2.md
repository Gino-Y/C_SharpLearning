# 委托练习 2：带参数的委托（游戏伤害播报）

> [!NOTE]
> 练习 1 的委托无参无返回，这次练**带参数**的委托。
> 前置：[delegate 委托](./委托.md) · [委托练习 1](./委托-练习1.md)

## 场景

游戏里 Boss 每次受到攻击，要把「伤害数值」广播给多个关注者：血条 UI 要刷新、战斗日志要记录、成就系统要累计。

## 要求

### 1. 声明带参数的委托

```csharp
public delegate void DamageHandler(int damage);   // 装「接收一个 int、无返回值」的方法
```

### 2. Boss 类

- 委托变量 `OnDamaged`（可空）
- 方法 `TakeDamage(int damage)`：打印 `Boss 受到 {damage} 点伤害`，然后把 `damage` 传给委托广播出去

### 3. 三个关注者类（方法签名都必须是 `void 方法名(int)`）

- `HealthBar`（血条）：`Refresh(int damage)` → 打印 `血条减少 {damage}`
- `BattleLog`（日志）：`Record(int damage)` → 打印 `日志:本次伤害 {damage}`
- `Achievement`（成就）：`Accumulate(int damage)` → **累计**总伤害到自己的字段里,并打印 `累计伤害 {总量}`

> 考点一：委托带参数时，调用委托要传实参：`OnDamaged?.Invoke(damage)`。
> 考点二：`Achievement` 有**自己的状态**（累计值字段）——体会「注册的方法属于某个对象，能访问该对象的数据」。

### 4. `Run()` 方法

1. 创建 Boss 和三个关注者，全部 `+=` 注册
2. 连续攻击三次：`TakeDamage(100)`、`TakeDamage(250)`、`TakeDamage(50)`
3. 观察成就系统的累计值是否一直在涨（100 → 350 → 400）

## 预期输出

```
Boss 受到 100 点伤害
血条减少 100
日志:本次伤害 100
累计伤害 100
Boss 受到 250 点伤害
血条减少 250
日志:本次伤害 250
累计伤害 350
Boss 受到 50 点伤害
血条减少 50
日志:本次伤害 50
累计伤害 400
```

## 提示（卡住再看）

- 声明委托时括号里写参数：`delegate void DamageHandler(int damage);`
- 内置写法 `Action<int>` 完全等价，写完自定义版后可以试着换成它。
