# delegate 委托

[下一篇：委托练习 1](./委托-练习1.md)

> [!NOTE]
> 对应 Lesson19a（耦合版）与 Lesson19b（委托版）。委托 = 能装方法的变量。
> 前置：[方法与方法重载](../Class01-基础语法/方法与方法重载.md) · [类与对象](./类与对象.md) · 相关枢纽：[委托体系](../枢纽/枢纽_委托体系.md)

## 委托写法（Lesson19b）

```csharp
public delegate void ClassOverHandler();
public ClassOverHandler? OnClassOver;
public void ClassOver() => OnClassOver?.Invoke();

controller.OnClassOver += room.ClassOver;  // 注册多播
controller.ClassOver();                    // 广播
```

| | 非委托 | 委托 |
|---|---|---|
| 字段 | `List<ClassRoom>` | 委托变量 |
| 认识教室？ | 必须 | 否，谁 `+=` 谁收通知 |

## 要点

- 声明：`delegate 返回类型 名(参数);`；签名必须匹配。
- 调用前判空：`?.Invoke()`；`+=`/`-=` 注册/取消。
- 内置：`Action`/`Action<T>`、`Func<T,TResult>`；event 建立在委托之上。

> [!TIP]
> 类比微信群（`+=` 进群，Invoke 群发）。Python 函数可直接当对象；C# 委托把签名类型化。
