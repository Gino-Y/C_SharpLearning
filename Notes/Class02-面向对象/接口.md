# 接口 interface

[上一篇：继承与多态](./继承与多态.md) · [下一篇：泛型](./泛型.md)

> [!NOTE]
> 对应 `IHotChair.cs`、`Lesson07b` 中 `Audi`。前置：[继承与多态](./继承与多态.md)

```csharp
public interface IHotChair { void HotChair(); }   // 只声明，默认 public
class Audi : CarA, IHotChair {                    // 基类在前，接口在后
    public void HotChair() => Console.WriteLine("座椅加热");
}
```

## 要点

- 契约：必须有哪些成员，不含实现（传统）。名以 `I` 开头。
- 可实现**多个**接口，弥补单继承；实现方法须 `public`。
- 多态：`IHotChair x = new Audi();`

| | interface | abstract class |
|---|---|---|
| 实现 | 传统上无 | 可有 |
| 数量 | 多个 | 一个基类 |
| 用途 | 附加「能做什么」 | 「是什么」+ 共享代码 |
