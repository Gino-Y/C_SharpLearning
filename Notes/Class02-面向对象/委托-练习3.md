# 委托练习 3：委托当参数传递（自制筛选器）

[上一篇：委托练习 2](./委托-练习2.md)

> [!NOTE]
> 把方法当参数传。前置：[委托](./委托.md) · [练习2](./委托-练习2.md) · 相关枢纽：[委托体系](../枢纽/枢纽_委托体系.md)

## 要求

1. `delegate bool ScoreFilter(int score);`
2. `static List<int> Filter(List<int> scores, ScoreFilter condition)` — 遍历，`condition(score)` 为真则收入。
3. `IsPass`（≥60）、`IsExcellent`（≥90）。
4. **Run**：成绩 `{55,78,92,45,88,96,61}`；`Filter(..., IsPass/IsExcellent)`；Lambda 筛不及格：`score => score < 60`。

## 预期

```
及格: 78 92 88 96 61
优秀: 92 96
不及格: 55 45
```

提示：传方法名不加括号；此即 LINQ `Where` 的底层形态（`Func<int,bool>`）。
