# 类与对象

> 对应 `Class02/Lesson01.cs`

```csharp
class Student
{
    // 属性（自动属性 get/set）
    public string Name { get; set; }
    public int Age { get; set; }
    public string Gender { get; set; }
}

// 使用：
Student student = new Student();   // new 一个对象（实例）
student.Name = "张三";              // 用「对象.成员」赋值
student.Age = 18;
Console.WriteLine(student.Name);
```

## 要点

- **类 (class)** = 模板/图纸；**对象 (object)** = 用 `new` 造出的具体实例。
- 每个对象的属性**互相独立**：`student` 和 `student2` 的 `Name` 互不影响。
- `public string Name { get; set; }` 是**自动属性**：编译器自动生成读(get)/写(set)。
- 类里放两类东西：**数据**（字段/属性）和 **行为**（方法）。
- Python 对照：`__init__` 里的 `self.x` ≈ C# 实例字段/属性。
