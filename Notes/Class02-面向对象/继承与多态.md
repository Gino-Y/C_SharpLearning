# 继承、多态、抽象类

> [!NOTE]
> 对应 `Class02/Lesson07.cs`、`Lesson07b.cs`（完整代码见课程文件）
> 前置：[类与对象](./类与对象.md) · [构造函数](./构造函数.md) · 相关：[接口 interface](./接口.md)（多来源的补充方案）

## 继承

```csharp
public class Cat : Animal   // 冒号读作「继承自」，自动拥有 Animal 的成员
```

- C# **只能单继承**（冒号后只能一个父类）；多来源用 [接口](./接口.md)。
- 子类自动获得父类的 `public`/`protected` 成员。

## virtual / override / abstract

```csharp
public abstract class Animal      // 抽象类：不能被 new
{
    public void Eat() { }             // 普通方法：不能被 override
    public abstract void Sleep();     // 抽象方法：无实现，子类【必须】重写
    public virtual void Walk() { }    // 虚方法：有默认实现，子类【可以】重写
}

public class Cat : Animal
{
    public override void Sleep() { }  // 必须实现抽象方法
    public override void Walk() { }   // 可选重写虚方法
}
```

| 关键字 | 位置 | 有方法体 | 含义 |
|--------|------|---------|------|
| `virtual` | 父类方法 | ✅ 有默认实现 | 允许子类重写，不重写用默认 |
| `abstract` | 父类方法（类也须 abstract） | ❌ 无 | 强制子类重写 |
| `override` | 子类方法 | ✅ | 重写父类的 virtual/abstract |

- 要 `override`，父类方法**必须**是 `virtual` 或 `abstract`；普通方法不能被重写。
- **抽象类**不能直接 `new`，只能被继承。
- `new`（方法隐藏）≠ 重写：`override` 看对象真实类型，`new` 看变量声明类型；要多态用 `virtual`+`override`。

## 多态 / 里氏转换原则

```csharp
Animal mimi = new Cat();   // 父类变量指向子类对象（里氏转换）
mimi.Sleep();              // 运行时执行 Cat 的 Sleep（动态分派）

static void BeginCar(CarA car) { car.Begin(); }   // Lesson07b：父类参数接任意子类
BeginCar(new Audi());      // 执行 Audi 的 Begin
```

- **里氏转换原则**：子类对象可以赋给父类类型的变量。
- **多态**：通过父类变量调 virtual/abstract 方法，运行时执行**对象真实类型**的版本。
- 好处：用统一的父类类型处理不同子类，无需逐个 `if` 判断类型。

> [!TIP]
> Python 对照：Python 方法默认可重写（无需 `virtual`/`override`）；抽象方法用 `@abstractmethod`。
