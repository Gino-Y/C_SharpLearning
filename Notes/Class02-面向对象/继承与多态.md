## 继承

```csharp
public class Cat : Animal   // 冒号 = 继承自；自动拥有父类 public/protected 成员
```

C# **只能单继承**；多来源用 [接口](./接口.md)。

## virtual / override / abstract

```csharp
public abstract class Animal {           // 不能 new
    public abstract void Sleep();        // 无实现，子类必须 override
    public virtual void Walk() { }       // 有默认，子类可 override
}
public class Cat : Animal {
    public override void Sleep() { }
    public override void Walk() { }
}
```

| 关键字 | 有方法体 | 含义 |
|--------|---------|------|
| `virtual` | 有默认 | 允许重写 |
| `abstract` | 无（类也须 abstract） | 强制重写 |
| `override` | 有 | 重写 virtual/abstract |

- 普通方法不能 `override`；`new`（隐藏）≠ 重写——要多态用 `virtual`+`override`。

## 多态 / 里氏转换

```csharp
Animal mimi = new Cat();   // 父类变量 → 子类对象
mimi.Sleep();              // 运行时走 Cat.Sleep
BeginCar(new Audi());      // 父类参数接任意子类（Lesson07b）
```

- **里氏**：子类可赋给父类类型。**多态**：父类变量调 virtual/abstract，执行真实类型版本。

> [!TIP]
> Python：方法默认可重写；抽象用 `@abstractmethod`。
