## 语法

`访问修饰符 返回值 this[参数…] { get{} set{} }` — 完整见课例。

```csharp
public Person this[int index] { get { … } set { … } }     // 管 friends
public int this[int i, int j] { get => array[i,j]; set => array[i,j]=value; }
public string this[string str] { get { /* name/age */ } } // 可只读
p[0] = new Person(); Console.WriteLine(p["name"]);
```

## 要点

- get/set 可写逻辑（规则自定）；可按参数类型/数量重载；struct 也支持。
- 适合「类本身像一个集合」；多个并列集合更宜暴露为属性。

> [!WARNING]
> - 字段未 `new` → NullReferenceException → 声明时初始化。
> - 越界分支穿透 → 正常写入放 `else`。

> [!TIP]
> Python ≈ `__getitem__`/`__setitem__`；C# 多了重载与编译期检查。
