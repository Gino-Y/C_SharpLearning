# enum 枚举

> 对应 `Class02/Lesson18.cs`。枚举 = **一组有名字的常量**（值类型），和 struct 一样属于「类型定义」，不是集合。

```csharp
// 定义：成员默认对应 0,1,2,3...（也可手动指定，如 Spring = 1）
enum Season
{
    Spring,   // 0
    Summer,   // 1
    Autumn,   // 2
    Winter    // 3
}

// 使用
Season now = Season.Summer;
Console.WriteLine(now);          // Summer，打印的是名字
Console.WriteLine((int)now);     // 1，底层是整数

// 整数 ↔ 枚举 互转
Season s = (Season)2;            // 2 → Autumn

// 常和 switch 搭配：取值固定，分支清晰
switch (now)
{
    case Season.Spring: ... break;
    case Season.Summer: ... break;
    default: ... break;
}
```

## 为什么用枚举

- 代替「魔法数字」：`if (season == 1)` 看不懂；`if (now == Season.Summer)` 一目了然。
- **编译器把关**：`Season x = 5;` 直接报错（要强转才行），非法值进不来。
- 取值固定的场景都适合：季节、星期、状态（订单状态、游戏状态）、方向等。

## 要点

- 枚举是**值类型**，底层默认是 `int`，成员从 0 开始编号。
- 可手动指定值：`enum Season { Spring = 1, Summer = 2, ... }`。
- 强转：`(int)枚举值` 取数字，`(枚举类型)数字` 转回枚举。
- 最佳拍档是 `switch`（取值有限、分支清晰）。

## Python 对照

```python
from enum import Enum
class Season(Enum):
    SPRING = 0
    SUMMER = 1
```

C# 的 enum 是语言内置关键字，更轻量；取底层值 C# 用 `(int)x`，Python 用 `x.value`。
