# 内部类与分部类

> [!NOTE]
> 对应 `TLS_HX/Lesson11_封装_内部类和分部类.cs`（唐老师核心课程）。内部类 = 在一个类中再申明一个类；分部类 = 用 `partial` 把一个类分成几部分申明。
> 前置：[类与对象](./类与对象.md)

## 内部类

- **概念**：在一个类中再申明一个类。
- **特点**：使用时要用**包裹者点出自己**，如 `Person.Body`、`Person.Body.Arm`。
- **作用**：亲密关系的体现（Body 只为 Person 服务）。
- **注意**：访问修饰符作用很大——内部类不 `public`，外面就点不出来。

```csharp
class Person
{
    public Body body;                     // 只是声明字段，没有实例化

    public class Body                     // 内部类
    {
        public Arm leftArm;
        public class Arm { }              // 内部类还能再套内部类
    }
}

// 使用：包裹者点出自己
p.body = new Person.Body();
p.body.leftArm = new Person.Body.Arm();
```

## 分部类（partial）

- **概念**：把一个类分成几部分申明，关键字 `partial`，编译时合并成一个类。
- **作用**：分部描述一个类；增加程序的拓展性（常见于自动生成代码 + 手写代码分离）。
- **注意**：可以写在**多个脚本文件**中；各部分访问修饰符要**一致**；**不能有重复成员**。

```csharp
partial class Student { public bool sex; }      // 第一部分
partial class Student { public int number; }    // 第二部分，合并后两个字段都有
```

## 分部方法（局限性大，了解即可）

- **概念**：将方法的**申明和实现分离**，写在分部类的两个部分里（`partial void Speak();` / `partial void Speak() { ... }`）。
- **特点**：不能加访问修饰符（默认私有）；只能在分部类中申明；返回值只能是 `void`；可以有参数但不能用 `out`。

> [!TIP]
> Python 对照：内部类 Python 也能写（类体里再定义类，访问方式同样是 `Person.Body`）；分部类无对应物——Python 一个类只能在一处定义完。
